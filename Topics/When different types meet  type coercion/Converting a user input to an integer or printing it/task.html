Converting a user input to an integer or printing it

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Kotlin program that receives an input value. If the input is a numeric string, it should convert it to an integer. If the input is a floating-point number, it should convert it to an integer through truncation (removing the fractional part). In case the input is a non-numeric string or an integer, it should print it as it is. In your implementation, avoid using conditional statements or loops.
Sample Input:
123.45

Sample Output:
123


Sample Input:
456

Sample Output:
456



Memory limit: 256 MB
Time limit: 8 seconds
